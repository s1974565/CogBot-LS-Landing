Reach users — one tap to a call

From the dashboard, send notifications: dialogue invitations or task reminders. Tapping one opens a VoIP-style incoming-call screen, so a gentle reminder turns straight into a real conversation. Reminders can be scheduled to go out on their own.

Tune prompts against simulated dialogue

Designing a good companion prompt is usually manual trial and error. CogBot makes it a workflow — define personas, scenarios, and evaluation criteria, then generate simulated conversations between CogBot and each persona. Revise the prompt automatically from those runs, or by hand. Personas give feedback too, and the same loop sharpens the personas themselves.

Two-stage loop — build realistic personas first, then refine the prompt against them.
